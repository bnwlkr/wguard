## Installation 

To install wguard, run:

```
brew tap bnwlkr/wguard
brew install wguard
```


## Usage

### Block sites

```
sudo wguard -b facebook.com youtube.com
```

### Unblock sites

```
sudo wguard -u facebook.com youtube.com
```

### List Blocked Sites


```
wguard -l
```
